# Change Log

All notable changes to the "branch-note" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

## [0.0.2] - 2025-12-14

### Fixed

- Fixed issue where note history was not showing instantly after saving new notes
- Tree view now refreshes automatically when new notes are saved
- Fixed variable scope issue that prevented proper view refreshing

### Added

- Automatic tree view refresh after note operations
- Improved note history display responsiveness

## [0.0.1] - 2025-12-14

### Added

- Initial release of Branch Note extension
- Create and view branch-specific notes
- Rich text editor with formatting toolbar
- Note history tracking per branch
- Branch monitoring and automatic note notifications
- Delete individual notes or entire branch history
